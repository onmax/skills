# Writing web documentation

This skill bundle helps you write top-notch developer documentation for web projects.

## Contents

- `SKILL.md` — the main workflow and quality bar
- `assets/` — ready-to-use templates and checklists
- `references/research-notes.md` — research synthesis from strong documentation sites and style guides

## Best use cases

Use this bundle when you are creating or improving:

- a project README
- a docs homepage
- a quickstart
- a tutorial
- a how-to guide
- API or SDK reference
- concept / architecture docs
- troubleshooting docs
- migration or upgrade guides

## Suggested workflow

1. Start with `assets/documentation-brief-template.md`.
2. Choose the right page type.
3. Draft with the matching template.
4. Apply the rules in `SKILL.md`.
5. Review with `assets/review-checklist.md`.
